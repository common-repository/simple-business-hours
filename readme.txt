Simple Business Hours plugin allows blog admin to display business working hours.

== Description ==

Simple Business Hours plugin allows to admin display its business working hours in any sidebar area using a widget.


= Features =

* Customizable Widget Title
* Time Format Setting
* Set Start Day of Your Business
* Mutilingual (Translation Ready)


== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip archive and upload the entire folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate Widget `Simple Business Hours` to display business hour in any sidebar in your theme.
